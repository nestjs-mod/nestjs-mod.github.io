# Тесты: Обзор

## Обзор

Этот раздел содержит сгенерированную документацию по тестовым сценариям `nestjs-mod`.
На каждой странице есть ссылка на исходный тест, ключевой подготовительный код и описание проверяемого поведения.

## Структура

- `common/config-model` - трансформация и валидация модели конфигурации
- `common/env-model` - парсинг и валидация переменных окружения
- `common/nest-application` - поведение bootstrap и lifecycle
- `common/nest-module` - композиция модулей и DI-контракты
- `common/project-utils` - утилиты проекта и инфраструктурные helper-сценарии
- `apps/example-basic` - сценарии базового приложения
- `apps/example-fastify` - сценарии с адаптером Fastify
- `apps/example-tcp-microservice` - сценарии взаимодействия TCP-микросервисов
- `integrations/*` - интеграционные тестовые сценарии

## Как использовать

1. Откройте нужную категорию в sidebar.
2. Выберите страницу сценария.
3. Перейдите по GitHub-ссылке к точному месту теста.
4. Сверьте setup и assertions с вашей реализацией.

## Примечания

- Заголовки и подписи нормализованы для читаемости.
- `id` сделаны уникальными, чтобы избежать конфликтов Docusaurus.
- Технические промежуточные пути вроде `src/app` убраны из навигации.
